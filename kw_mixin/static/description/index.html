Kitworks Base Mixins


Technical module that provides a set of reusable mixins for Odoo models. These mixins enhance model functionality with various date-related operations, string processing, and utility methods.

Key features include:

  • Date Parts Processing: Year, Week, Month, Quarter operations
  • Day of Week Utilities: Handling and name conversions
  • String Processing: Transliteration and cleaning methods
  • Date Extraction: Methods to extract dates from various formats

Available Mixins

Date Operations

  • Year Mixin:
    • Year-based operations and computations
    • Fiscal year handling
  • Week Mixin:
    • Week number handling
    • Week-based calculations
  • Month Mixin:
    • Month-based operations
    • Month start/end date calculations
  • Quarter Mixin:
    • Quarter-based operations
    • Quarter validations

String Processing

  • Transliteration:
    • General transliteration methods
    • KMU rules-based transliteration
    • String cleaning utilities

Date Extraction

  • Format Support:
    • String to date/datetime conversion
    • Integer timestamp handling
    • Multiple format parsing
